이미지 다운로드 11.1GB VirtualBox Disk Image ROS_1.vdi



VirtualBox로 ROS 빠른 설치하기

1. VirtualBox를 실행 시킨뒤, 새로 만들기를 클릭한다.




2. 이름(임의), 머신 폴더(기본값 권장), 종류(Linux), 버전(Ubuntu x64)을 선택하고 다음을 클릭한다.




3. 메모리 용량(임의)을 지정해준다. 본인은 2048MB를 지정해주었다.




4. 기존 가상 하드디스크 파일 사용을 선택하고 불러오기를 클릭한다.




5. 추가 버튼을 클릭한다.




6. 미리 다운로드 받은 ROS_1.vdi 파일을 선택한다.




7. ROS_1.vdi 파일이 선택되었는지 확인 후, 선택 버튼을 클릭한다.




8. 만들기를 클릭한다.



9. 선택사항 기본적으로 VirtualBox에서 Linux 머신을 만들때, 1 Core를 사용한다. Core 수를 조절 하려면, 다음과 같이 따라한다.




9-1. 선택사항 시스템 -> 프로세서 -> 프로세서 개수를 조절하고 확인 버튼을 클릭한다.




10. 시작 버튼을 클릭한다.




11. 시스템이 로딩 되는것을 기다린다.




12. 계속 기다린다...(정상작동시 까지 5~10분 소요)




13. 로딩이 완료되면 비밀번호를 입력하는 칸이 뜨는데, 비밀번호는 're1199re'이다.


보안을 위해 비밀번호를 변경하는것을 권장함!


14. 바탕화면이 표시되면 마우스 오른쪽 버튼을 클릭해 'Open Terminal'을 클릭한다.




15. 터미널에 'roscore'라고 타이핑 하고 엔터를 누른뒤, ROS가 정상 작동하는지 확인한다.
